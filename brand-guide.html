This guide defines the visual language of Sunai (سنيع) — a three-sided event management platform connecting clients, vendors, and administrators across Saudi Arabia. The brand is built on the calligraphy wordmark: pure ink black for authority and precision, forest green for growth, trust, and the richness of Saudi event culture. Every token in this document is the single source of truth for the product UI.

05 — Motion

Duration & Easing

Duration

fast
150ms — hover, icon swaps
normal
250ms — card lift, panel open
slow
400ms — page enter, modal in
slower
600ms — skeleton-to-content reveal

Easing Curves

default

cubic-bezier(0.4, 0, 0.2, 1)

Standard ease-in-out for most transitions

spring

cubic-bezier(0.34, 1.56, 0.64, 1)

Slight overshoot — buttons, cards, badges

easeOut

cubic-bezier(0, 0, 0.2, 1)

Enter animations — elements sliding in

easeIn

cubic-bezier(0.4, 0, 1, 1)

Exit animations — elements leaving view

Principle: Motion should feel purposeful and never decorative. Use fast for micro-interactions (hover feedback), normal for layout shifts, slow/slower only for screen-level transitions. Keep all transitions under 600ms to respect users with prefers-reduced-motion.

Do's & Don'ts

✓ Do

  • Use jet-black for primary CTAs
  • Use forest green for accent / active states
  • Maintain clear space around the logo
  • Keep the logo on white, black, or forest surfaces
  • Use Almarai for all Arabic text

✗ Don't

  • Never stretch or distort the logo
  • Never use low-contrast color combinations
  • Never mix purple, orange, or gold with the brand
  • Never place the logo on a busy photograph
  • Never use font weights other than 400 or 700
